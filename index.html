Project Steps

The

The first step involved importing the layoffs dataset into MySQL. I ensured the data was properly loaded without errors, using the appropriate file format and encoding.

Step 2: Data Cleaning

In this step, I removed duplicates, standardized column formats, addressed null values, and removed unnecessary columns to prepare the data for analysis.

Step 3: Exploratory Data Analysis

I analyzed the cleaned data to identify trends, patterns, and insights into layoffs globally. Visualization tools were used to support the analysis.
